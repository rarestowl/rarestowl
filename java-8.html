Anonymous Inner Class vs Lambda Expression:
If anonymous inner class implements an Interface that contains single abstract method then only we can replace that anonymous inner class with lambda expressions.
Anonymous Inner Class can extend normal class, abstract class and interface with multiple abstract methods.

Default Methods(inside interface): without affecting implementation classes if we want to add new method to the interface.
Every variable is public static final.
Until 1.7, every method present inside interface always public and abstract.
1.8, static methods + default methods
1.9, private methods
default method also known as virtual extension method or defender method

                                       public void m1();
